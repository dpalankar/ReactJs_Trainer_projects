Adding features to Javascript

One of the most powerful features of V8 is that it can be embedded into any other C++ program. It provides hooks for accessing its core feature set.

It might sound ordinary at first, but think about it - Now you can write functions and extend the functionality of Javascript. Think about it - JS was not supposed to deal with files or other system related stuff. It was supposed to run in a browser. But, with this feature, we can make it do things that are beyond the scope of ECMA specifications
