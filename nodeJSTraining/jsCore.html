The Javascript core

Look at the 'lib' folder. It has JS files. These are essentially some functionalities provided to you right out of the box. These are simply wrappers. Let's look at zlib.js. Check the function - process.binding

This is essentially going and grabbing the C++ library. So, when you use the zlib function, you use the JS file which, in-turn, uses grabs the C++ file

Basically, node has gone out of its way to ensure that using the utilities is as easy for you as possible. Also, not all functions here are embedding C++ functions. For example, look at the util.js. It is a simple library with some important utility functions written in JS
