The command line interface

We normally use the GUI to perform our day to day operations in our operating system. There is another way to do this - the CLI. You type the commands in the CLI rather than click in GUI

  • Linux

    Bash

  • Mac

    Terminal

  • Windows

    Command prompt

You can execute commands to check the list of folders or copy or do other stuff.

Arguments to commands

You can also pass arguments to your commands. Example:

dir /w

dir /all

We can also pass multiple arguments if the command supports it. Example:

dir /w /p

Windows powershell
